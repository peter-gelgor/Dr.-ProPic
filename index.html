How does it work?


Dr. ProPic is a state-of-the-art technology which rates your attractiveness out of 10 to help pick which photo would attract the most dimes. We use AI and human judgement to ensure that we can accurately tell you that you should hit the gym and/or a plastic surgeon.

Using cutting-edge technology and Microsoft Azure's machine learning model creation tools, we developed a neural network to classify and quantify an attractiveness rating to be assigned to a person based on
user-submitted photos.

Based on training data from an intricate data set of 1000 faces and their attractiveness ratings, our machine learning model classifies pictures based on Microsoft's Vision API to design a program to quantify subjective attractiveness ratings over a set of criteria.
